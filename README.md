# ASP.NET Core MVC CRUD Application
A simple CRUD application built with ASP.NET Core 8, MVC, EF Core (Code First), and SQL Server.  
Demonstrates basic Create, Read, Update, Delete operations in a clean MVC architecture.

## Features
- Create, Read, Update, Delete records
- MVC architecture
- EF Core Code First approach
- SQL Server integration

## Tech Stack
- ASP.NET Core 8, C#, EF Core, SQL Server

## How to Run
Open the project in Visual Studio or VS Code, update the database connection string, and launch the application.
